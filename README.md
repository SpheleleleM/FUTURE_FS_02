# Mini CRM - Client Lead Management System

A full stack web application built with Node.js, Express, SQLite and vanilla JavaScript.

## What it does
- Leads submit their details through a public contact form
- Admin logs into a secure dashboard to manage all leads
- Admin can update lead status (new → contacted → converted)
- Admin can add follow-up notes to each lead
- Dashboard shows live stats and supports search and filtering

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** bcrypt, express-session

## How to run locally

1. Clone the repository
   git clone https://github.com/YourUsername/FUTURE_FS_02.git

2. Install dependencies
   npm install

3. Seed the database with sample data
   node seed.js

4. Start the server
   node server.js

5. Open your browser and go to
   http://localhost:3000
